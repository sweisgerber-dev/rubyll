# Rubyll

> A ruby-based docker image for building, verifying and and publishing websites.


## Tools:

> Debian Stretch

- python 3.5 (default)
- python 2.7
- pip
- npm
- nodejs
- build-essentials
- ftp
- imagemagick


## [Docker Hub](https://hub.docker.com/r/sweisgerber/rubyll/)


## Build

```bash
docker build -t rubyll .
```

## Run

```
docker run -i -t rubyll
```
